#pinOrUnpin

Based on currently focused SmartButton, we'll decide we are going to pin or unpin the app. This function is invoked when we get 'click' event on pinToHomeElem. Notice that we use MozActivity to pin app because we want to bring users to Home when they finish pinning. However we unpin app via IAC because we want users stay in AppDeck after unpinning.

#sendMessage(type, data)

Send message on 'appdeck-channel' using Inter-App Communication. smart-home will monitor this channel. Currently we use this channel to tell smart-home to what app we would like to unpin.

AppDeck#focus-on-pinable

This event fires whenever focus in AppDeck move to a pinable element (representing na app).

Properties

  • Boolean pinned :

    Is current focused pinable element pinned or not
  • String manifestURL :

    manifestURL of current focused element
  • String name :

    name of current focused pinable element
  • Boolean removable :

    Is current focused pinable element removable or not

fireFocusEvent(elem)

Notify other module that currently focused element is pinable (could be pinned on Home) or nonpinable (could not be pinned on Home)

Parameters

  • HTMLElement elem :

    currently focused element
