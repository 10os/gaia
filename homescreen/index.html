AUTOSCROLL_DELAY

The timeout before auto-scrolling a page when hovering at the edges of the grid.

AUTOSCROLL_DISTANCE

The distance at the top and bottom of the icon container that when hovering an icon in will cause scrolling.

AUTOSCROLL_OVERFLOW_DELAY

The time to wait after setting a scroll-position before disabling overflow during drag-and-drop.

BLACKLIST

Strings that are matched against to black-list app origins. TODO: This should not be hard-coded.

COLUMNS_SETTING

Name of the global home screen columns setting.

datastore

The datastore

db

Our local indexed db where we store our copy of bookmarks

DB_VERSION

The version of the indexed database

lastRevision

The last datastore revision that was synced

name

The name of the datastore

DIALOG_SHOW_TIMEOUT

Timeout before showing a dialog. Without this, the click that comes through after an activate event from gaia-container will close the dialog.

HASH_CHANGE_DEBOUNCE

Time after receiving a handled hashchange that we ignore hashchange events for.

HIDDEN_ROLES

App roles that will be skipped on the homescreen.

PAGES_ICON_SIZE

The size in pixels of the icons in the pinned pages.

PAGING_SETTING

Name of the global home screen paging setting.

RESIZE_TIMEOUT

Timeout before resizing the apps grid after apps change.

SETTINGS_STORE

Name of the global home screen settings datastore.

SETTINGS_VERSION

Stored settings version, for use when changing/refactoring settings storage.
