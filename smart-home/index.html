FilterManager

FilterManager is the controller class of filter. It relies on CardFilter to do UI works of filter itself. However it also needs to co-operate with SmartBubbles, Edit, and Home classes to do the actual filtering task in smart-home.

The working flow when user change filter is like:

  1. FilterManager receives filterchanged from CardFilter
  2. Play bubbling down animation, see onFilterChanged().
  3. When animation ends, change card icon in smart-home, with help from Home and XScrollable. See onAnimationEnd().
  4. Play bubbling up animation, see _performBubbleUp().

#onAnimationEnd

EventHandler of all-items-bubbled event. SmartBubbles fires all-items-bubbled when it is done with bubbling down animation. We start to alter the card content of XScrollable only after bubbling down animation ends. And after we are done with altering content, we'll have to perform bubbling up animation by calling FilterManager#_performBubbleUp.
